# AKSARA UI DESIGN SYSTEM

## 1. Design Philosophy
Aksara menggunakan pendekatan **Modern Minimalist + Glassmorphism + Bento UI** untuk menciptakan LMS yang bersih, profesional, cepat dipahami, dan nyaman digunakan.

- Professional
- Educational
- Modern SaaS
- Friendly
- Accessible
